图形学中常用三角形作为基础图元去表示其他复杂的形状 Fundamental Shape Primitives

选择三角形 why？

  • 是最基本的图形;复杂的图形可以分解成三角形。
  • 三角形一定是在一个平面上的;可以用叉乘快速判断点在三角形的内外关系；可以用重心坐标插值

Drawing a Triangle To The Framebuffer (“Rasterization光栅化”) 光栅化=绘图到光栅显示器

  • What？输入：投射到屏幕上的三角形顶点位置。输出：一组近似三角形的像素值。这就是我们要做的事情。

  • How？通过sampling采样。

采样sampling

  • what：将连续的函数离散化的过程 We can discretize a function by sampling.即一个连续的函数，我们在各个不同的地方去取它的值，这个过程就被叫做采样Evaluating a function at a point is sampling.

Rasterization 光栅化

光栅化的本质就是对屏幕空间做 2D采样。那么对于光栅化一个三角形来说，采样就是判断像素的中心是否在三角形内

inside(tri,x,y)可以用来表示这个像素点是否在三角形内。1表示在里面。

那么可以通过采样函数来进行三角形的光栅化 即 f(x,y) = inside(tri,x,y)

快速判断左右，简单来说就是用向量的叉乘。右手螺旋定则，结果为正说明在左侧

那么如何判断像素的中心是否在三角形内？ 用三次同方向的叉乘。

但是我们没有必要判断所有的pixel是否在三角形内。用boudingbox来减少判断的范围。用三角形的三个点来得到这个包围盒的范围。对于沿斜对角放置的瘦长型对象优化效果并不好。

Lecture 3: Intro to Signal Processing:Sampling, Aliasing, Antialiasing

Sampling is Ubiquitous（广泛存在的） in Computer Graphics

  • Rasterization = Sample 2D Positions
  • Photograph = Sample Image Sensor Plane
  • Video = Sample Time

接下来看一下常见的走样所形成的问题Artifacts ：Sampling Artifacts in Graphics and Imaging

  • Jaggies（锯齿），空间中采样

  • Moire（摩尔纹），对图片做降采样（把图片中的奇数列和奇数行给去掉）

  • Wagon wheel effect（车轮效应），时间中采样（常见于高速行驶的车辆，人眼观察车轮会有向后转的假象）

总结一下这些问题，走样的本质就是信号的变换太快（高频），采样的频率太慢（采样间隔大）

fast-changing signals (high frequency), when we sample too slowly

这里我们先给出反走样的方法：采样之前先做模糊（去除高频信号） Antialiasing Idea: Filter Out High Frequencies Before Sampling

以光栅化三角形为例：先对三角形做一次模糊（去掉高频的信号），把它变成一个边界模糊的三角形，采样边界时，离三角形边界近的地方像素颜色深一些，远的地方像素颜色浅一些。

接下来解释为什么做完模糊再采样可以减少走样现象。

how to implement antialiased rasterization

采样的理论基础 可跳过 这部分更多是信号与系统学的，应该不会考。简单看一下就行。总之我们把信号从时域转换到频域用来分析为什么 去除高频信号（模糊）后再采样 可以减少走样现象。

傅里叶变换可以将函数表示为正弦和余弦的加权和 Represent a function as a weighted sum of sines and cosines

空间域/时域和频域可以通过傅里叶变换和逆傅里叶变换相互转换 Fourier Transform Decomposes A Signal Into Frequencies

傅里叶变换把信号从时域（spatial domain）转换到频域（frequency domain）

why？傅里叶变换可以把信号从时域转化到频域，因为从时域或空间域的角度看，该信号是由多种频率不同的正弦波的叠加，而从频域角度看，该信号的频谱却如此的简单，这便于我们的分析信号。总之傅里叶可以帮助我们把时域信号变成频域以便于信号分析。

  • Higher Frequencies Need Faster Sampling高频需要更快的采样才能保证恢复出的函数更接近原始函数。在固定的采样频率下，信号的频率越高，采样出来的结果就越差。
  • Undersampling Creates Frequency Aliases在相同的采样率下去采样频率差异巨大的两种函数，却得到了相同的结果，导致我们无法区分这两种信号，这种现象成为”走样“ Two frequencies that are indistinguishable at a given sampling rate are called “aliases”

滤波就是去掉某些特定频率的信号（Filtering = Getting rid of certain frequency contents）

前面我们提到反走样就是在采样之前先做一次模糊，把高频的信号给去掉，这个去掉某些频率信号的过程被叫做滤波（Filtering）。

现在我们借助前面提到的频域来理解滤波的过程和结果。傅里叶变换将时域变为频域：低频集中在中间，高频信息在外围(边缘处)

  • 高通滤波（得到边界） High-pass filt (Edges) 去除低频信息 —> 边缘特征

    • 只有高频信息可以通过，后将频谱图逆傅里叶变换得到图像

  • 低通滤波（得到模糊效果） Low-pass filter (Blur) 去除高频信息 –> 模糊处理（存在水波纹）

    • 只有低频信息可以通过

Filtering = Convolution 也许可以是计算。其实就是加权平均计算

  • 卷积定理

    • 时域的卷积等于频域的乘积，频域的卷积等于时域的乘积（两者具有对偶关系） Convolution in the spatial domain is equal to multiplication in the frequency domain, and vice versa

  • 通过卷积定理实现对图像的处理

    • 方法一： 简单来说 加权平均

      • 在时域空间域中通过卷积过滤 Filter by convolution in the spatial domain

    • 方法二： 在频域上做乘积

      • 通过傅里叶变换将时域转化为频域 Transform to frequency domain (Fourier transform)
      • 乘以卷积核的傅里叶变换 Multiply by Fourier transform of convolution kernel
      • 通过逆傅里叶变换将频域转化回时域 Transform back to spatial domain (inverse Fourier)

  • 图像中小的正方体对应低通滤波器Box Function = “Low Pass” Filter。卷积核越大，在图像上进行滤波会越模糊，其本身的频率越低Wider Filter Kernel = Lower Frequencies

Sampling（采样） = Repeating Frequency Contents（重复频域上的内容）

Things to Remember

Signal processing key concepts: 信号处理关键概念

  • Frequency domain vs spatial domain 频域和时域
  • Filters in the frequency domain scale frequencies 滤波
  • Filters in the sampling domain = convolution 卷积
    Sampling and aliasing 采样和走样
  • Image generation involves sampling 用采样生成图片
  • Nyquist frequency is half the sampling rate
  • Frequencies above Nyquist appear as aliasing artifacts
  • Antialiasing = filter out high frequencies before sampling 如何反走样：在采样前过滤高频信号
  • Interpret supersampling as (approx) box pre-filter antialiasing

LinearAlgebra lec02

lec02这节课是图形学需要用到的线代知识的复习，比较简单，这里只简要列出，主要是要知道在图形学的应用。
vector：向量的加法、点乘dot product、叉乘cross product。点乘得到一个数值，叉乘得到一个新的向量。
matrix：矩阵加法、矩阵和向量相乘、矩阵和矩阵相乘、转置、单位矩阵、逆矩阵、伴随矩阵

点乘：${
{ \overrightarrow{a} \cdot \overrightarrow{b} }
= { {\lVert \overrightarrow{ a } \rVert} {\lVert \overrightarrow{ b } \rVert} {\cos\theta} }
}$ 点乘性质：交换律、结合律、分配律

叉乘：${
{ \lVert \overrightarrow{a} \times \overrightarrow{b} \rVert }
= { \lVert \overrightarrow{a} \rVert }{ \lVert \overrightarrow{b} \rVert }\sin\theta
}$ ；右手螺旋判断方向。叉乘结果的模在大小上等于两个向量组成的三角形面积

矩阵性质：结合律、分配律、没有交换律。

  • $ (AB)^T = {B^T}{A^T} $
  • $AA^{-1}= A^{-1}A=I$
  • $(AB)^{-1}=B^{-1}A^{-1}$

在图形学上的应用：

  • 点乘
    • 求任意两个向量之间的夹角（例如光源在表面之间角度的余弦）。
    • 求一个向量在另一个向量上的投影
    • 根据结果正负判断一个向量对于另一个向量是朝前还是朝后的。
  • 叉乘
    • 判断向量在另一个向量的左边还是右边 ； ${
      { \overrightarrow{a} \times \overrightarrow{b} >0 }
      }$ 表示b在a左边
    • 判断点P在三角形内还是外（三次按顺序的叉乘方向相同的话在内）
  • 矩阵
    • 常用矩阵来表示变换信息：平移、旋转、缩放、切变等 Translation, rotation, shear, scale

Transformation lec03+lec04部分

先具体讲几种2D的变换：Scale Transform 缩放1变换矩阵、Reflection Matrix对称2变换矩阵、Shear Matrix切变3变换矩阵、Rotation Matrix旋转4变换矩阵。我们发现以上四种都可以用一个矩阵乘以输入可以得到输出的坐标，这种叫做线性变换。平移也是常用的变换，但不是线性变换。为了不特殊处理平移的情况，引入齐次坐标Homogeneous coordinates来统一处理各种变换，具体做法是增加一个维度。
然后还有6仿射变换Affine和7逆变换。仿射= 线性变换+平移
然后3D是在lec04的，也放在这里，与2D类似，不具体讲。

  1. 缩放
  2. 对称
  3. 切变：竖直方向是没有变的/y不变，沿水平拉伸。
  4. 旋转矩阵的逆就是它的转置：正交矩阵 $A^{-1} = A^T$

    这几种都是线性变换： 输出坐标 = 矩阵 乘以 输入坐标

齐次坐标

why？平移不能用线性变换表示，但是我们又不想把它作为特殊情况处理。所以引入齐次坐标，把所有的变换写成一个矩阵乘以一个向量。

  2. 逆变换

通过组合可以表示复杂的变换。反过来也可把复杂的变换分解。要注意顺序（因为做变换其实就是左乘一个矩阵，矩阵的乘法是不满足交换律的）

Blinn-Phong Reflection Model: 三种光照的组合

$$\begin{split}
L &= L_a + L_d + L_s \
&= k_aI_a + k_d(I/r^2)max(0, \mathbf{n} \cdot \mathbf{l}) + k_s(I/r^2){max(0, \mathbf{n \cdot h})}^p
\end{split}$$

  • 环境光Ambient lighting $L_a = k_aI_a$ 给模型所有位置均匀的光照，这里简单认为是常数，其实是不对的。
  • 漫反射光Diffuse Reflection，光线打到物体的表面会被均匀的散射到各个方向
    • $L_d = k_d(I/r^2)max(0, \mathbf{n} \cdot\mathbf{l})$
    • 光线照射到模型的方向与模型法线，夹角越小的地方，光照越强
  • 镜面反射光/高光Specular highlights，观察方向和光线的镜面反射方向接近的地方，模型会产生高光
    • $$\begin{split}
      L_s &= k_s(I/r^2){max(0, \cos\alpha)}^p \
      &= k_s(I/r^2){max(0, \mathbf{n \cdot h})}^p
      \end{split}$$

几个k分别是对应的系数

着色频率：逐三角形、逐顶点、逐像素。几何足够复杂的情况下，用逐三角形也可以达到好的效果。

实时渲染管线：3D场景到2D图的过程

  • Vertex Processing
    主要是把我们输入的顶点进行MVP变换，得到屏幕空间的顶点表示，方便后续处理。

  • Triangle Processing
    按照一定顺序，将顶点组成不同的三角形

  • Rasterization
    对每个像素进行采样，判断像素在不在三角形内

  • Fragment Processing -> Z-buffer 、 Shading、Texture mapping

    计算shading point（根据不同的着色频率决定不同粒度的shading point）的颜色，并着色
    这个阶段也会做纹理映射，简单说就是把贴图（颜色、法线等等）给映射到物体的不同位置上，后面会详细说明这一部分

  • Framebuffer Operations
    ​ 根据像素大小，最终生成一个用于显示的Framebuffer
