LinearAlgebra lec02

lec02这节课是图形学需要用到的线代知识的复习，比较简单，这里只简要列出，重点是要知道线代在图形学的应用。向量的加法、点乘、叉乘。矩阵加法、乘法。

点乘：${
{ \overrightarrow{a} \cdot \overrightarrow{b} }
= { {\lVert \overrightarrow{ a } \rVert} {\lVert \overrightarrow{ b } \rVert} {\cos\theta} }
}$
性质：交换律、结合律、分配律

点乘应用：

  • 求任意两个向量之间的夹角（例如光源在表面之间角度的余弦）。
  • 求一个向量在另一个向量上的投影
  • 判断两个向量方向是否一致。判断向量相较于另一向量是向前还是向后

叉乘

  • 判断向量在另一个向量的左边还是右边 正左边
  • 判断点P在三角形内还是外（三次按顺序的叉乘）

图形学中常用矩阵来表示变换信息：平移、旋转、缩放、切变等 Translation, rotation, shear, scale
性质：结合律、分配律、没有交换律
矩阵和向量相乘
矩阵和矩阵相乘
转置、单位矩阵、逆矩阵、伴随矩阵
